
# ClubGPT

A chatbot that works with GPT Chat for use in Clubhouse software chat rooms.

## Features

- Reusable chat-bot
- Typescript
- Dockerize
- Customizable

## Installation

Install ChartGPT with Yarn

```bash
  yarn
  
  yarn start
```
